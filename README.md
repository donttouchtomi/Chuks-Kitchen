## Project Description

This project is a modern shopping website interface designed to simulate a real-world online store.

It includes:

- Product listing section
- Add to cart functionality
- Dynamic cart total calculation
- Responsive navigation with hamburger menu
- Clean and structured UI design

The goal of this project was to strengthen my understanding of front-end development and improve my JavaScript skills.

### Technolgies Used

- HTML5
- CSS3
- JavaScript
- Netlify

## Features

- Fully responsive design (mobile, tablet, desktop)
- Interactive hamburger menu
- Add items to cart
- Automatically updates total cart price
- Organized and reusable layout structure
- Clean user interface

### How It Works

## Navigation

The hamburger button uses JavaScript to toggle the visibility of the mobile navigation menu.

## Cart System

- Users can click “Add to Cart”
- Items are added dynamically
- The total price updates automatically using JavaScript
